# Using the CLI

rtmf6 provides a command-line interface.

The simplest way is to change to directory with the configuration file
 `rtmf6.toml` and type:

```bash
rtmf6
```

Alternatively, you provide the path the configuration file:

```bash
rtmf6 run path/to/rtmf6.toml
```

The command:

```bash
rtmf6
```

is equivalent to:

```bash
rtmf6 run
```
